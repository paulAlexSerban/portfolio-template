# Atoms

- The foundational design elements that you cannot break down further. For example, buttons, icons, forms, etc.
- if you can't divide without it becoming useless, then it's an atom

## Possible Atoms

- badge
- checkbox
  - control with three states: unselected, selected, and indeterminate (situation where a list of sub-options is grouped under a parent option and sub-options are in both selected and unselected states)
- input-field
- input-label
- link
- pill
- radio-button
- slider
- tab
- tag
- text
- toggle-switch
  - represents a physical switch that allows users to turn things on or off, like a light switch
- tooltip
