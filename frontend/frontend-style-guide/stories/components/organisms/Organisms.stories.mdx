# Organisms
- Complex UI patterns comprising of atoms and molecules. These patterns shape a user interface with cards, navigation bars, logos, search fields, etc.
- Combinations of multiple molecules and have more than one function
- Organisms let us break down the web-page into a series of smaller applications - we can craft pages or screens out of a series of components

## Possible Organisms
- carousel
- dialog
- footer
- form
- light-box
- media-grid
- music-player
- navigation-bar
- progress-bar
- tab-bar
- table
- video-player
