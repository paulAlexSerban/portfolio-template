# Molecules
- Created by combining atoms to create larger UI components or patterns, like pagination, breadcrumbs, etc.
- groups of atoms that work as a single component with a single function

## Possible Molecules
- blockquote
- breadcrumbs
- button-groups
- card
- date-picker
- dropdown-button
- dropdown-menu
- loading
- media-objects
- modal
  - alert
  - boolean - Eg. cookies: accept/cancel
- notice
- notification
- pagination
- popup
  - error
  - informative
- search
- tile
- time-picker
