# Viewports and Breakpoints

## Breakpoints

Breakpoints are customizable widths that determine how your responsive layout behaves across device or viewport sizes.

### Core concepts

- Breakpoints are the building blocks of responsive design. Use them to control when your layout can be adapted at a particular viewport or device size.

- Use media queries to architect your CSS by breakpoint. Media queries are a feature of CSS that allow you to conditionally apply styles based on a set of browser and operating system parameters. We most commonly use min-width in our media queries.

- Mobile first, responsive design is the goal.

### Available breakpoints and viewpoorts

| Breakpoint    | Viewport             | Dimensions | Media Query                                 | Widths              |
| ------------- | -------------------- | ---------- | ------------------------------------------- | ------------------- |
| `$mobile-xs`  | smartwatch-only      | <`319px`   | `@media (max-width: 319px), (hover: none)`  | 0 - `319px`         |
|               | smartphone           | >`320px`   | `@media (min-width: 319px), (hover: none)`  | `320px` - `479px`   |
| `$mobile`     | smartphone-landscape | >`480px`   | `@media (min-width: 479px), (hover: none)`  | `480px` - `959px`   |
| `$tablet`     | tablet               | >`960px`   | `@media (min-width: 959px), (hover: none)`  | `960px` - `1439px`  |
|               | desktop-small        |            | `@media (min-width: 959px)`                 |                     |
| `$desktop`    | desktop-touch        | >`1440px`  | `@media (min-width: 1439px), (hover: none)` | `1440px` - `1919px` |
|               | desktop              |            | `@media (min-width: 1439px)`                |                     |
| `$ultra-wide` | ultra-wide           | > `1920px` | `@media (min-width: 1919px)`                | `1920+px`           |

### How to use

Set properties per breakpoints

- _scss source_

```css
.example {
  @include breakpoint($tablet) {
    color: red;
  }
}
```

- _css output_

```css
@media only screen and (min-width: 959px) {
  .example {
    color: red;
  }
}
```

Set properties for specific viewports

- _scss source_

```css
.breakpoint-example {
  @include viewport(smartphone-landscape) {
    color: red;
  }
}
```

- _css output_

```css
@media only screen and (min-width: 479px), only screen and (hover: none) {
  .example {
    color: red;
  }
}
```
